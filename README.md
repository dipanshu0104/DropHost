# What is DropHost? 

The DropHost server is basically used to create a self hosted file uploading server. Which is use
to upload a file on a server computer with the help of a impressive web UI.

## Features

 - Easy to setup.
 - User friendly interface.
 - Connect multiple devices.
 - Send the data to the server your server.

# Installation and requirments

- This tool require NodeJS, and also ExpressJS and Multer module. After you can use it.

## Installation

- Step 1. Clone the code by git clone. 

```bash  
git clone https://github.com/dipanshu0104/DropHost.git 
```
( You can also download the zip file directly. )

- Step 2. First download the node  packages like Express, Multer and nodemon by these commands to go at the drophost folder path, And run these commands.

```bash  
npm install -g nodemon 
npm install express multer
```

- Step 3. Now run the server.js file code using nodemon or you can run directly.

### Run using nodemon
```bash  
   npx nodemon server.js
```
### Run using node
```bash  
   node server.js
```
